here is a code in c language that returns root of a number:
```c
#include <stdio.h>
#include <math.h>

int main(){
  int number = 0;
  scanf("%d", &number);
  printf("%f",pow(number, 0.5));
  return 0;
}
```  
here is a test and answer :
```c
input = 4

answer = 2.0000
```
